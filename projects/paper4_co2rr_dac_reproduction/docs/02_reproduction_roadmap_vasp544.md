# VASP 5.4.4 复现路线

## 1. 总策略

复现分成三层，不一次性提交几百个作业。

### 层级 A：软件和基本功复现

使用 GitHub 的 NiZn-N-C 输入，学会：

- POSCAR、POTCAR、INCAR、KPOINTS；
- VASP 5.4.4 超算提交；
- 电子收敛与离子收敛；
- OUTCAR、OSZICAR、CONTCAR 检查；
- 磁矩、能量和最大力提取。

这一层验证“会正确使用软件”，不验证论文最终结论。

### 层级 B：代表性论文结果复现

优先复现：

- NiMn-N-C 裸位点；
- `*COOH`、`*CO`、`*H`；
- SnNi-N-C 裸位点；
- `*OCHO`、`*HCOOH`、`*H`；
- 两个形成能以及关键吸附自由能。

这一层用于验证 NiMn 偏 CO、SnNi 偏 HCOOH 的核心结论。

### 层级 C：完整 91 体系筛选

只有层级 B 的能量和趋势与论文足够接近后，才批量生成 91 个体系和全部中间体。

## 2. GitHub 项目与论文的差异

| 项目 | 论文 | GitHub 示例 | 处理方式 |
|---|---|---|---|
| 金属集合 | 13 种，含 Sn | 27 种过渡金属，不含 Sn | 重新建立论文元素表 |
| 组合数 | 91，包含同核 | 主要针对异核组合 | 增加 M1=M2 |
| 示例中心 | NiMn、SnNi | NiZn | NiZn 只做入门 |
| HCOOH 中间体 | OCHO、HCOOH | 缺少 OCHO | 新增构型生成 |
| 磁态 | 多初始磁矩比较 | 每种元素单一初值 | 建立磁态扫描 |
| slab 弛豫 | 正文称原子位置弛豫 | 吸附脚本冻结全部 slab | 代表性复现允许位点附近原子弛豫，并核对 SI |
| 预弛豫 | 未报告 CHGNet | 使用 CHGNet | 仅作初猜，不作为论文方法 |
| VASP 结果 | 原文有完整数据 | 仓库没有 OUTCAR/CONTCAR | 必须自行计算 |
| 自由能 | SI 有完整修正 | 参考能文件缺失 | 取得 SI 后重建 |

## 3. VASP 5.4.4 的特别约束

### 3.1 不使用 LATTICE_CONSTRAINTS

`LATTICE_CONSTRAINTS` 是较新 VASP 才有的功能，VASP 5.4.4 不可使用。

### 3.2 含真空石墨烯不要直接 ISIF=3

仓库文档对含 15 A 真空的石墨烯使用 `ISIF=3` 优化晶胞，这会同时放开真空方向，可能使 c 轴异常变化。

VASP 5.4.4 的兼容方法：

1. 固定 `c = 15-20 A`；
2. 构建一系列平面晶格常数，例如 `a = 2.43-2.51 A`；
3. 每个 a 对原子位置优化或做一致的静态计算；
4. 绘制 `E(a)`；
5. 用二次或状态方程拟合最低点；
6. 用所得 a 建立超胞。

### 3.3 必须确认 VASPsol 编译状态

虽然论文设置 `LSOL = .TRUE.`，但超算上的 `vasp_std` 必须包含相应 VASPsol 代码。

需要向管理员确认：

```text
1. VASP 5.4.4 是否替换/更新了 solvation.F？
2. VASPsol 的具体版本是什么？
3. 是否使用 -Dsol_compat 编译选项？
4. 是否有经过验证的水溶剂测试算例？
```

首次运行后还应检查 OUTCAR 是否明确读取了 solvation 参数，而不是只看 INCAR 中写了 `LSOL`。

## 5. 阶段 0：超算基线测试

用一个小体系确认：

- `vasp_std` 能启动；
- PAW-PBE 赝势库可用；
- MPI 与 NCORE 设置兼容；
- VASPsol 确实启用；
- 作业不会因磁盘配额写满而中止。

对于论文的 3x3x1 K 点，使用 `vasp_std`，不要使用只适合单 Gamma 点的 `vasp_gam`。

## 6. 阶段 1：数值收敛测试

选择 NiMn-N-C 或 NiZn-N-C 作为代表结构，固定几何，分别测试：

### ENCUT

```text
400, 450, 500, 550, 600 eV
```

### K 点

```text
2x2x1, 3x3x1, 4x4x1, 5x5x1
```

### 真空

```text
15, 18, 20 A
```

判断时优先比较能量差、吸附能和原子力，而不是比较巨大总能量的绝对值。建议目标是关键能量差变化小于约 0.02-0.05 eV；最终采用阈值应记录在项目日志中。

## 7. 阶段 2：参考态

形成能和自由能要求所有参考态使用相容设置。

需要计算或核对：

- 石墨烯每个 C 的能量；
- N 的参考态（正文只写 `nitrogen`，SI 也没有进一步定义，仍需向作者确认或清楚声明工作假设）；
- 13 种金属最稳定体相每原子能量；
- H2、H2O、CO、CO2、HCOOH 等气相参考；
- 振动频率、ZPE 和熵修正。

不能把不同 POTCAR 版本、不同 ENCUT 或不同溶剂条件的能量直接相减。

## 8. 阶段 3：裸双原子位点

### 8.1 结构模型

正文和 SI 都无法唯一确定以下细节，必须从作者结构数据核对或自行重建：

- 超胞到底是 4x4、5x5 还是 6x6；
- 精确的 6N 配位拓扑；
- 两个金属的初始高度和相对位置；
- 是否存在多个缺陷异构体。

GitHub 的 6x6、62C+6N+2M 模型可作为候选初始模型；SI Figure S5 只提供吸附构型图片，没有坐标，因此该模型仍不能自动视为论文原始结构。

### 8.2 磁态扫描

论文明确测试不同初始磁矩。对 NiMn 至少准备多组初猜，例如：

```text
Ni/Mn = 0/0, 1/3, 2/5, 2/-5, -2/5
```

这些只是初始条件，不是最终磁矩。每组都要优化并比较最终能量、局域磁矩和是否收敛。最低能、结构合理的解才进入后续吸附计算。

## 9. 阶段 4：吸附构型搜索

每种中间体不能只放一个方向。至少要考虑：

- 吸附在 M1；
- 吸附在 M2；
- 桥式跨两个金属；
- 不同分子朝向；
- 对 OCHO 考虑单齿和双齿；
- 对 COOH 考虑 C 端吸附及合理旋转；
- 不同初始磁态。

需要的核心物种：

```text
CO2, COOH, CO, OCHO, HCOOH, H
```

所有初始构型完成 VASP 优化后，选择最低能且化学结构合理的局部极小值。不能只因为某个作业最先收敛就采用它。

## 10. 阶段 5：Gibbs 自由能与筛选

对 CO 和 HCOOH 两条路径分别计算四个步骤的 DeltaG，并得到：

```text
DeltaGmax_CO
DeltaGmax_HCOOH
DeltaG_HER
```

按 SI，CHE 后处理至少包含 `DeltaE`、`DeltaZPE`、电位项 `qU`、pH 项、热容积分和熵项。论文取 `pH=0`；吸附中间体的热修正来自振动频率，气相分子的焓熵来自 NIST-JANAF。SI 未给出逐物种修正表，所以本项目必须保存频率输出和自行生成的修正表，不能直接套用来历不明的常数。

随后按论文逻辑：

1. 使用形成能评价稳定性；
2. 使用 DeltaGmax 评价活性；
3. 与 HER 比较评价选择性；
4. 计算 `E*COOH` 和 `E*OCHO` 与火山顶的距离；
5. 复核 NiMn 和 SnNi 的排名。

`results/reference_table_s2.csv` 保存了 SI Table S2 的 91 行原始数据。该表前三个印刷列名与正文数值存在循环错位的强证据；分析脚本必须保留原始列名，并另设显式的解释映射，禁止直接覆盖原始数据。

## 11. 阶段 6：电子结构和机理

代表体系可继续计算：

- 总 DOS 和金属 d-PDOS；
- 吸附原子 p-PDOS；
- d 带中心；
- 差分电荷密度；
- Bader 电荷（如课题组已有工具）；
- Ni-Mn、Sn-O、Mn-C/O 等轨道杂化区间。

PDOS 计算应在充分优化的几何上使用更严格的静态自洽设置，并单独做 K 点和展宽检查。

## 12. 阶段 7：完整筛选和机器学习

在代表体系复现成功后：

1. 构造 91 个裸位点；
2. 扫描磁态；
3. 计算形成能；
4. 对保留体系计算关键中间体；
5. 构建 20 个描述符；
6. 复现八种回归模型；
7. 比较 R2、RMSE、MAE；
8. 用 SHAP 检查 `E*COOH` 是否仍为主导特征；
9. 复现 -0.51 eV 和 -0.43 eV 两个火山顶。

严格按 SI 先采用 4:1 随机训练/测试拆分，训练集内做 5 折 CV，先随机搜索、再网格搜索。由于 SI 没有提供随机种子、样本拆分、搜索范围或最终超参数，我们自己的实现必须固定随机种子、导出样本编号和搜索结果；若指标与论文略有差异，应报告分布或多种子结果，而不是人为挑选一次最接近的拆分。

完整 91 体系后再做高成本验证：SnNi 的 NVT AIMD（300 K、20 ps）和 NiMn/ZnMn/IrMn 的四层显式水对照。SI 未给出 AIMD 时间步、恒温器或完整 INCAR，这部分只能在明确工作假设后做方法级复现。

## 13. 第一阶段的完成标准

在进入完整论文复现前，应先拿到一个 NiZn-N-C 或 NiMn-N-C 裸位点的完整结果包：

```text
INCAR
POSCAR
POTCAR 哈希或 TITEL 记录（不要公开 POTCAR 内容）
KPOINTS
提交脚本
vasp.out
OUTCAR
OSZICAR
CONTCAR
vasprun.xml
```

并证明：

- 每个离子步电子收敛；
- 结构达到力阈值；
- OUTCAR 正常结束；
- 元素和 POTCAR 顺序正确；
- VASPsol 正确启用；
- 最终结构没有金属飞离位点或原子异常重叠；
- 最终磁矩被记录。
